# Human（个人模型）

> 所有人生决策的统一分析框架。

## 继承关系

```text
System
  └── LifeSystem
        └── Human
```

继承 System 的 9 个属性 + LifeSystem 的 8 个属性，新增个人特有的维度。

---

## 属性

| # | 属性 | 含义 |
|---|------|------|
| 1 | 身体（Body） | 健康、精力、外貌 |
| 2 | 意识（Consciousness） | 认知能力、思维方式、元认知 |
| 3 | 资源（Resources） | 金钱、资产、社会关系 |
| 4 | 时间（Time） | 寿命、可支配时间 |
| 5 | 能力（Ability） | 技能、知识、经验 |
| 6 | 信息（Information） | 掌握的知识、情报、数据 |
| 7 | 价值观（Values） | 什么重要、什么不重要 |
| 8 | 目标（Goal） | 短期/长期追求的状态 |
| 9 | 环境（Environment） | 家庭、社会、时代背景 |

---

## 推导结论

所有人生决策，本质都是改变以上属性的状态：

| 决策 | 改变的属性 |
|------|-----------|
| 学习 | 能力 ↑、信息 ↑ |
| 投资 | 资源 ↑（或 ↓） |
| 健身 | 身体 ↑ |
| 旅行 | 信息 ↑、资源 ↓ |
| 社交 | 资源（社会关系）↑、信息 ↑ |
| 跳槽 | 资源 ↑、环境改变 |
| 使用 AI | 能力 ↑（扩展） |
| 结婚 | 环境改变、资源合并 |
| 生育 | 资源 ↓、时间 ↓、目标改变 |

---

## 核心矛盾

- 时间有限 vs 目标无限 → 必须排序 → 价值观
- 资源有限 vs 欲望无限 → 必须取舍 → Trade-off
- 能力有限 vs 环境复杂 → 必须学习 → 持续演化
- 身体衰减 vs 时间不可逆 → 必须规划 → 长期思维
